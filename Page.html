Правила работы

Начало работы
Выбираем пункт меню <Замены> - Замены - Замены Правила Замен 1, соответственно (2,3,4,5). Все, работает.
Добавление правил
Правила замен могут быть добавлены самостоятельно. Для этого открываем Google Disk и в корне Диска находим файл _!_Pravila1 (2,3,4,5). Например, для пункта меню "Правила 3", файл будет _!_Pravila3 и т.д. Отрываем этот файл и чез разделитель @ пишем правило. Например правило кораблья@машина найдет слова корабль и заменит их на слова - машина.
Создание правил
Для начала работы входим в меню Змены -> Системная - > Тебе туда надо? ->
Выбираем для какого пункта создаем файл замен и создаем файлы с правилами Замен. Созданные файлы появятся на Google Disk, в корне Диска,имена файлов будут _!_Pravila1 (2,3,4,5).
Например, для пункта меню "Правила 3", файл будет _!_Pravila3 и т.д.
При формировании новых файлов правила формируются только для 1 и 5 файла, остальные файлы формируются с пустым шаблоном.
Чтобы внести новые правила находим соответственно файл _!_Pravila1 (2,3,4,5), открываем его и вносим новые правила через разделитель @. Например правило кораблья@машина найдет слова корабль и заменит их на слова - машина.
